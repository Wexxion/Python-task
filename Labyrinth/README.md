Запуск программы прозводится из файла Main.
Для запуска консольной версии запустить solver и передать
на вход txt файл с картой который лежит по умолчанию в текущией директории проекта,
альфа параметр(в моем случае не от [0:1], a [0:100], так проще), кол-во бомб.
пример запуска:
	1)консольный режим : >python main.py -m console -f Resources/maze1.txt -b 5 -a 70
	2)графический режим: >python main.py -m gui


Результат работы программы - путь в лабиринте, учитывая альфу и кол-во доступных бомб.
Усложнения: + альфа параметр 
			+ несколько выходов
			+ порталы
			+ однопроходные стенки
			
Что бы создать карту надо запустить приложение с помощью команды >python main.py -m gui
и выбрать размеры лабиринта. Если выбранные размеры карты не влезут на экран, то программа 
не запустится пока не будут указаны размеры поменьше.
В окне редактора есть сообщения подсказки - верхний левый угол. При нажатии на кнопку туда будет
выведена короткая инструкция как пользоваться кнопкой либо будут выведены заметки. 
Кнопка Add start позволяет добавить стартовую позицию на карту.
Кнопка Add Finishes позволяет добавить несколько финишей на карту.
Кнопка Add Portals позволяет добавить порталы. Что бы создать пару порталов нужно левым кликом поставить
	первый портал, потом правым кликом мыши поставить второй портал.
One-way Wall (правый верхний угол) - мини меню, клкнув на него будет выбор типа однопроходной стенки.
	У данной стенки специальная отрисовка. Часть клетки белого цвета означает что с этой строны можно
	пройти через стенку не использую бомбы. Черная часть этой стенки означает что с этой строны прохода только с бомбой.
	
Снизу кнопка Save Map позволит вам сохранить карту с данными параметрами. Для того что бы указать кол-во бомб
	нужно зайти в solver и выставить там бомбы. Сохраненный лабиринт представляет из себя саму карту, где
	'#' -стена, ' ' -свободная клекта, а так же, после карты идет информация о бомбах, местоположениях старта, финиша,
	масштаба карты.
Кнопка Solver - открывает диалоговое окно, в нем можно задать кол-во бомб, параметр отрисовки пути, альфа-параметр,
	увидеть прогресс нахождения пути. Чем больше бомб - тем медленнее поиск пути.
	Что будет написано в подсказках:
		1) При введенных параметрах путь не был найден 
		2) Путь был найден, информация о длине пути и использованных бомбах.
Кнопка Continue - убрать отрисованный путь и продолжить редактирование карты.
Кнопка Exit - выход из редактора.


Декомпозиция:
	1)Components
		I)GUI
			a)Main_MapEditor.py - основное окно, точка запуска GUI
			b)MazeWidget.py - виджет для отображения лаюбиринта
			c)ParamsWindow.py - диалоговое окно для открытия файла и создания карты
			d)SolverThread.py - тред для выполнения алгоритма
			e)SolverWindow.py - диалоговое окно для выбора параметров для алгоритма
		II)maze.py - класс лабиринта с сохранением в файл и его открытием
		III)solver.py - сам алгоритм для решения лабиринта, точка запуска консольной версии
		IV)extra.py - класс вершины для алгоритма + класс с направлениями для однопроходных стенок
			
	2)Resources - папка с картами
	3)main.py - запуск программы, пакетный режим для консольной версии.
	4)Test - папка с файлами для тестов
	5)tests.py - сами тесты
