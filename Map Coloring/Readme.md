Запуск программы прозводится из файла Main.
Для запуска консольной версии запустить solver и передать
на вход txt файл с картой который лежит по умолчанию в текущией директории проекта.
формат карты: '#' - стена ' ' - ничего, в конце файла на отдельной строке пишется масштаб карты
(пример: 5, 10, 15 пикселей)


Результат работы программы - раскрашеный граф-карта в соответсвии с уловиями задачи.



Декомпозиция:
	1)Components
		a)myMap.py - класс краты
		b)painter.py - алгоритмы раскраски
	2)MapEditor
		a)maps - папка с сохраненными картами
		b)screenshots - папка с сохраненными скриншотами карт
		c)editor.py - редактор карт с визуализацией
	3)Main.py - запуск программы, пакетный режим для консольной версии.
	4)Test - папка с файлами для тестов
	5)tests.py - сами тесты

Описание классов и функций:
	Map:
		Для содания может использоваться как текстовый файл, так и двумерный 
		самой карты.
		
		def find_countries - пробегается по всей крате и если наткнулся
			на пустой участок(не стену), то начинает поск в ширину пока
			не найдет точные границы данной замкнутой области. Это и будет 
			страна. Страны имеют названия в виде чисел, наичиная с 1. 0 - граница
			
		def find_neighbors - пробегается по всем границам карты и ищет в соседних 
			клетках страны с разными номерами. Если таких стран больше одной, то 
			эти страны считаются соседями. Так же, соседями считаются только те страны,
			между которыми только стенка толщины в ОДИНУ точку. Результат в форме 
			графа - словаря. Ключ - номер страны, значение - лист из номеров соседей.
	
	Painter:
		Может использоваться как консольная версия раскарски карты, для этого
		надо запустить файл и указать какой текстовый файл открыть для создания
		карты. Умеет выводить карту, на который видны отдельные страны(нужно большое 
		разрешение экрана для комфортного просмотра), а так же выводит результаты 
		раскраски двумя алгоритмами в виде словаря, где ключ - номер страны, значение - 
		- номер цвета.
		
		def graph_coloring_optimal(graph) - правильный алгорим раскраски, принемает на вход
			граф - словарь.
			
		def graph_coloring_approximate(graph) - обычный алгорим раскарски, работает не хуже чем
			оптимальный, но зачастую на сложных картах с большим количесвом соседей и стран 
			испольует бОльшее количесво цветов. На вход граф - словарь.
			
	Editor:
		Визуализация редактора карт на PyQt5. Левый клик мыши позволяет рисовать линии для создания
		ограниченных областей. Правый клик - убирает стенку в данной точке(аналог ластика).
		
		class ParamsWindow - диалоговое окно, запускается первым. Пользователь либо загружает
			файл с картой, либо выбирает размеры новой карты и размер одной клетки на карте
			в пикселях(scale).
		
		class MapEditor - главное окно, имеет кнопки с функциями, аналог мессаджбокса, где 
			пользователь может увидеть подсказки. По середине mapWidget. 
			Кнопки:
				1) Save Map - сохраняет текущую карту в файл.
				2) Paint Map - открывает окно с выбором алгоритма и потом происходит раскарска
					карты и вывод результата на экран.
				3) Continue Editing - позволяет после применения раскраски карты продолжить 
					ее редактирование с текущего состояния.
				4) Screenshot - делает скриншот mapWidget'a и сохраняет его в картинку .bmp
				5) Exit - выход из редактора.
				6) New Line - позваляет сбросить текущую точку и начать рисование линии из новой
					точки. Для удобства.
				7) Add regions/ Clear regions - позваляет добавить регион. Разные регионы - разным цветом.
					новый регион - клик по кнопке. Несколько владельцев на один регион - несколько разных
					цветов на данный регион. Есть кнопка для очистки всех регионов.
		
		class MapWidget - собсвенный виджет для отрисовки карты и ее изменения.
